# Matrix Theory

## What is a Matrix?

A matrix is a rectangular array of numbers.

Example:

\[
A=
\begin{bmatrix}
1 & 2\\
3 & 4
\end{bmatrix}
\]

Rows:

2

Columns:

2

Dimension:

\[
2\times2
\]

---

## Matrix Notation

A matrix is denoted using uppercase letters.

Example:

\[
A,B,C
\]

Elements are denoted:

\[
a\_{ij}
\]

where:

- i = row index
- j = column index

---

## Types of Matrices

### Row Matrix

\[
[1\quad2\quad3]
\]

---

### Column Matrix

\[
\begin{bmatrix}
1\\
2\\
3
\end{bmatrix}
\]

---

### Square Matrix

Same rows and columns.

\[
3\times3
\]

---

### Diagonal Matrix

\[
\begin{bmatrix}
1&0&0\\
0&2&0\\
0&0&3
\end{bmatrix}
\]

---

### Identity Matrix

\[
I=
\begin{bmatrix}
1&0&0\\
0&1&0\\
0&0&1
\end{bmatrix}
\]

---

### Zero Matrix

Every element is zero.

\[
\begin{bmatrix}
0&0\\
0&0
\end{bmatrix}
\]

---

## Why Matrices Exist

Matrices provide efficient ways to:

- Store data
- Transform data
- Solve systems of equations
- Represent neural network weights
