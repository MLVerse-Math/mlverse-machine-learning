# Vectors

## What is a Vector?

A vector is an ordered collection of numbers.

Unlike scalars, vectors possess:

- Magnitude
- Direction

Examples:

2D Vector:

v = [3,4]

3D Vector:

v = [2,5,7]

Vectors are used to represent:

- Position
- Velocity
- Features
- Embeddings
- States

in modern AI systems.

---

## Why Vectors Matter?

Everything in AI eventually becomes a vector.

Examples:

Image

[255, 128, 64, ...]

Text Embedding

[0.25, -0.67, 1.42, ...]

User Features

[Age, Income, Education]

---

## Geometric Interpretation

A vector can be viewed as an arrow.

The arrow has:

- Starting Point
- Length
- Direction

Example:

v = [3,4]

moves:

3 units in x direction

4 units in y direction

---

## Types of Vectors

### Row Vector

[1 2 3]

### Column Vector

[1
2
3]

### Zero Vector

[0 0 0]

### Unit Vector

Magnitude = 1

### Position Vector

Represents location in space

## Components of a Vector

Example:

v = [4,3]
