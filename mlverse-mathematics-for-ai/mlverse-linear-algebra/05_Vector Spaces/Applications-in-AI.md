# Vector Spaces in Artificial Intelligence

## Machine Learning

Every data point is represented as a vector.

Example:

\[
(Age, Salary, Experience)
\]

creates a point in a 3D vector space.

---

## Word Embeddings

Words are mapped into high-dimensional vector spaces.

Examples:

- King
- Queen
- Man
- Woman

Each word becomes a vector.

---

## Computer Vision

Images are transformed into embedding vectors.

These vectors exist in learned feature spaces.

---

## Transformers

Transformer models learn semantic vector spaces.

Example:

\[
Embedding\ Dimension = 768
\]

Each token is represented as a vector in:

\[
\mathbb{R}^{768}
\]

---

## Recommendation Systems

Users and products are embedded into a common vector space.

Similarity between vectors helps generate recommendations.

---

## Generative AI

Latent spaces used in image and text generation are vector spaces.

Models navigate these spaces to create new outputs.

---

## Deep Learning

Every hidden layer creates a new learned vector space representation.
